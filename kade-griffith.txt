    1. What is Semantic HTML? 
-Semantic HTML are HTML tags that have a clear purpose as to what their function is. This helps web crawlers uderstand what kind of content is where on a webpage

    2. What is HTML used for? 
-HTML is used to form the backbone of a website. It creates the structure and content for other languages to tie into.

    4. What is the h1 tag used for? How many times should I use it on a page?

    10. What is a class? 
-A class is an HTML attribute that allows you to alter any element with that assigned class, without altering other elements of the same type.

    11. What elements can I add a class to?
-Any HTML element can have a class added to it. It's a global attribute.

    12. How many times can I use the same class on a page?
